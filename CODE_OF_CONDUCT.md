# Code of Conduct

## Our Pledge

OpenClassBoard is a tool built for educators. We're committed to keeping this community welcoming and respectful for everyone — regardless of experience level, background, or identity.

## Our Standards

**Expected behavior:**
- Be kind and constructive in feedback
- Respect differing opinions and approaches
- Help newcomers get oriented
- Focus on what's best for teachers and students using this tool

**Unacceptable behavior:**
- Harassment, insults, or personal attacks
- Dismissing contributions without explanation
- Spam or off-topic promotion

## Enforcement

Issues can be reported by opening a GitHub Issue marked **[conduct]** or contacting the maintainer directly via [Discord](https://discord.gg/KDfRFZ9YFe).

Violations may result in comments being removed or contributors being blocked at the maintainer's discretion.
